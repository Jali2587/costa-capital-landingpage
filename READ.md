# Costa Capital - Landing Page

AI-powered landing page for Costa Capital real estate financing in Spain.

## Features
- Bilingual (Dutch/English)
- AI chatbot for customer inquiries
- Interactive financing calculator
- Optimized for Spanish market (Valencia, Costa Blanca)

## Tech Stack
- React
- Tailwind CSS
- Netlify Functions
- Anthropic Claude API

## Deployment
Deployed on Netlify with serverless functions.
